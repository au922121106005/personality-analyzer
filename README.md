# 🧠 Personality Type Analyzer

A simple Python command-line application that analyzes your personality type using a set of 25 interactive questions based on MBTI (Myers-Briggs Type Indicator) logic.

---

## 🚀 Features

* 25-question personality assessment
* MBTI-based result (e.g., INFP, ENTJ)
* Displays full meaning of each trait (I, E, S, N, T, F, J, P)
* Interactive CLI (A/B input)
* Clean and readable output

---

## 🧩 How It Works

The program:

1. Asks 25 questions
2. Tracks your answers
3. Calculates your personality type
4. Displays:

   * MBTI result
   * Meaning of each letter
   * Personality description

---

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/au922121106005/personality-analyzer.git
cd personality-analyzer
```

### 2. Run the program

```bash
python main.py
```
