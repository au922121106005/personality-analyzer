# 🧠 Personality Type Analyzer

A Python command-line application that analyzes your personality type using a set of 25 interactive questions based on MBTI (Myers-Briggs Type Indicator) logic.

---

## 🚀 Features

- ✅ 25-question personality assessment  
- ✅ MBTI-based result (e.g., INFP, ENTJ)  
- ✅ Displays full meaning of each trait (I, E, S, N, T, F, J, P)  
- ✅ Interactive CLI (user inputs A/B answers)  
- ✅ Clean and simple output  

---

## 🧩 How It Works

The program:
1. Asks 25 questions
2. Tracks your answers
3. Calculates your personality type
4. Displays:
   - Your MBTI result
   - Full form of each letter
   - Personality description
